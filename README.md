# Instruction

In order to start the project:

1. Clone the remote repository to your development environment

```sh
$ git clone https://github.com/Varenthein/wdp-1811-03.git
```

2. Install depedencies

```sh
$ cd wdp-1811-03
$ npm install
```

# Most important tasks
